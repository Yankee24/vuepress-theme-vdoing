    Flinkеёёи§Ғзҡ„йқўиҜ•йўҳ

    # 1. Flink-On-Yarnеёёи§Ғзҡ„жҸҗдәӨжЁЎејҸжңүе“ӘдәӣпјҢеҲҶеҲ«жңүд»Җд№ҲдјҳзјәзӮ№пјҹ

    flinkеңЁyarnжЁЎејҸдёӢиҝҗиЎҢпјҢжңүдёүз§Қд»»еҠЎжҸҗдәӨжЁЎејҸпјҢиө„жәҗж¶ҲиҖ—еҗ„дёҚзӣёеҗҢгҖӮ

    # 1.1 yarn-session

    yarn-sessionиҝҷз§Қж–№ејҸйңҖиҰҒе…ҲеҗҜеҠЁйӣҶзҫӨпјҢз„¶еҗҺеңЁжҸҗдәӨдҪңдёҡпјҢжҺҘзқҖдјҡеҗ‘yarnз”іиҜ·дёҖеқ—з©әй—ҙеҗҺпјҢиө„жәҗж°ёиҝңдҝқжҢҒдёҚеҸҳгҖӮеҰӮжһңиө„жәҗж»ЎдәҶпјҢдёӢдёҖдёӘе°ұд»»еҠЎе°ұж— жі•жҸҗдәӨпјҢеҸӘиғҪзӯүеҲ°yarnдёӯе…¶дёӯдёҖдёӘдҪңдёҡе®ҢжҲҗеҗҺпјҢйҮҠж”ҫдәҶиө„жәҗпјҢйӮЈдёӢдёҖдёӘдҪңдёҡжүҚдјҡжӯЈеёёжҸҗдәӨгҖӮжүҖжңүдҪңдёҡе…ұдә«иө„жәҗпјҢйҡ”зҰ»жҖ§е·®пјҢJVMиҙҹиҪҪз“¶йўҲпјҢmainж–№жі•еңЁе®ўжҲ·з«Ҝжү§иЎҢгҖӮйҖӮеҗҲжү§иЎҢж—¶й—ҙзҹӯпјҢйў‘з№Ғжү§иЎҢзҡ„зҹӯд»»еҠЎпјҢйӣҶзҫӨдёӯеҸӘжңүдёҖдёӘJobManagerпјҢеҸҰеӨ–Jobиў«йҡҸжңәеҲҶй…Қз»ҷTaskManagerгҖӮ

    # 1.2 yarn-per-job

    жҜҸдёӘдҪңдёҡеҚ•зӢ¬еҗҜеҠЁйӣҶзҫӨпјҢйҡ”зҰ»жҖ§еҘҪпјҢJVMиҙҹиҪҪеқҮиЎЎпјҢmainж–№жі•еңЁе®ўжҲ·з«Ҝжү§иЎҢгҖӮеңЁper-jobжЁЎејҸдёӢпјҢжҜҸдёӘJobйғҪжңүдёҖдёӘJobManagerпјҢжҜҸдёӘTaskManagerеҸӘжңүеҚ•дёӘJobгҖӮдёҖдёӘд»»еҠЎдјҡеҜ№еә”дёҖдёӘJobпјҢжҜҸжҸҗдәӨдёҖдёӘдҪңдёҡдјҡж №жҚ®иҮӘиә«зҡ„жғ…еҶөпјҢйғҪдјҡеҚ•зӢ¬еҗ‘yarnз”іиҜ·иө„жәҗпјҢзӢ¬дә«Dispatcherе’ҢResourceManagerпјҢжҢүйңҖжҺҘеҸ—иө„жәҗз”іиҜ·пјҢйҖӮеҗҲи§„жЁЎеӨ§й•ҝж—¶й—ҙиҝҗиЎҢзҡ„дҪңдёҡгҖӮ

    # 1.3 yarn-application

    mainж–№жі•еңЁJobManagerдёӯжү§иЎҢпјҢе…ҘеҸЈзӮ№дҪҚдәҺApplicationClusterEntryPointпјҢе®ўжҲ·з«ҜеҸӘйңҖиҰҒиҙҹиҙЈеҸ‘иө·йғЁзҪІиҜ·жұӮгҖӮиҝҷз§Қж–№ејҸеҮҸиҪ»дәҶе®ўжҲ·з«Ҝзҡ„еҺӢеҠӣпјҢйҒҝе…Қе®ўжҲ·з«Ҝиө„жәҗжҲҗдёәз“¶йўҲгҖӮ

    # 2. жҖҺд№ҲеҒҡеҺӢеҠӣжөӢиҜ•е’Ңзӣ‘жҺ§пјҹ

    • дә§з”ҹзҡ„ж•°жҚ®жөҒзҡ„йҖҹеәҰеҰӮжһңиҝҮеҝ«пјҢиҖҢдёӢжёёзҡ„з®—еӯҗж¶Ҳиҙ№дёҚиҝҮжқҘзҡ„иҜқпјҢдјҡдә§з”ҹиғҢеҺӢгҖӮиғҢеҺӢзҡ„зӣ‘жҺ§еҸҜд»ҘдҪҝз”ЁFlink Web UIжқҘеҸҜи§ҶеҢ–зӣ‘жҺ§MetricsпјҢдёҖж—ҰжҠҘиӯҰе°ұиғҪзҹҘйҒ“гҖӮдёҖиҲ¬жғ…еҶөдёӢеҸҜиғҪз”ұдәҺsinkиҝҷдёӘж“ҚдҪңз¬ҰжІЎжңүдјҳеҢ–еҘҪпјҢеҒҡдёҖдёӢдјҳеҢ–е°ұеҸҜд»ҘдәҶгҖӮ
    • и®ҫзҪ®watermarkзҡ„жңҖеӨ§е»¶иҝҹж—¶й—ҙиҝҷдёӘеҸӮж•°пјҢеҰӮжһңи®ҫзҪ®зҡ„иҝҮеӨ§пјҢеҸҜиғҪдјҡйҖ жҲҗеҶ…еӯҳзҡ„еҺӢеҠӣгҖӮеҸҜд»Ҙи®ҫзҪ®жңҖеӨ§е»¶иҝҹж—¶й—ҙе°ҸдёҖдәӣпјҢз„¶еҗҺжҠҠиҝҹеҲ°зҡ„е…ғзҙ еҸ‘йҖҒеҲ°жөӢиҫ“еҮәжөҒдёӯпјҢжҷҡдёҖзӮ№жӣҙж–°з»“жһңгҖӮ
    • иҝҳжңүе°ұжҳҜж»‘еҠЁзӘ—еҸЈзҡ„й•ҝеәҰеҰӮжһңиҝҮеӨ§пјҢиҖҢж»‘еҠЁи·қзҰ»еҫҲзҹӯзҡ„иҜқпјҢFlinkзҡ„жҖ§иғҪд№ҹдјҡдёӢйҷҚзҡ„еҺүе®ігҖӮеҸҜд»ҘйҖҡиҝҮеҲҶзүҮзҡ„ж–№жі•пјҢе°ҶжҜҸдёӘе…ғзҙ еҸӘеӯҳе…ҘдёҖдёӘвҖңйҮҚеҸ зӘ—еҸЈвҖқпјҢиҝҷж ·е°ұеҸҜд»ҘеҮҸе°‘зӘ—еҸЈеӨ„зҗҶдёӯзҠ¶жҖҒзҡ„еҶҷе…ҘгҖӮ

    # 3. FlinkжҳҜйҖҡиҝҮд»Җд№ҲжңәеҲ¶е®һзҺ°зҡ„иғҢеҺӢжңәеҲ¶пјҹ

    FlinkеңЁиҝҗиЎҢж—¶дё»иҰҒз”ұoperatorsе’ҢstreamsдёӨеӨ§жһ„д»¶з»„жҲҗгҖӮжҜҸдёӘoperatorдјҡж¶Ҳиҙ№дёӯй—ҙзҠ¶жҖҒзҡ„жөҒпјҢе№¶еңЁжөҒдёҠиҝӣиЎҢиҪ¬жҚўпјҢз„¶еҗҺз”ҹжҲҗж–°зҡ„жөҒгҖӮеҜ№дәҺFlinkзҡ„зҪ‘з»ңжңәеҲ¶дёҖз§ҚеҪўиұЎзҡ„зұ»жҜ”жҳҜпјҢFlinkдҪҝз”ЁдәҶй«ҳж•Ҳжңүз•Ңзҡ„еҲҶеёғејҸйҳ»еЎһйҳҹеҲ—пјҢе°ұеғҸJavaйҖҡиҝҮзҡ„йҳ»еЎһйҳҹеҲ—пјҲBlockingQueueпјүдёҖж ·гҖӮдҪҝз”ЁBlockingQueueзҡ„иҜқпјҢдёҖдёӘиҫғж…ўзҡ„жҺҘеҸ—иҖ…дјҡйҷҚдҪҺеҸ‘йҖҒиҖ…зҡ„еҸ‘йҖҒйҖҹзҺҮпјҢеӣ дёәдёҖж—ҰйҳҹеҲ—ж»ЎдәҶпјҲжңүз•ҢйҳҹеҲ—пјүеҸ‘йҖҒиҖ…дјҡиў«йҳ»еЎһгҖӮ

    еңЁFlinkдёӯпјҢиҝҷдәӣеҲҶеёғејҸйҳ»еЎһйҳҹеҲ—е°ұжҳҜиҝҷдәӣйҖ»иҫ‘жөҒпјҢиҖҢйҳҹеҲ—е®№йҮҸйҖҡиҝҮзј“еҶІжұ пјҲLocalBufferPoolпјүе®һзҺ°зҡ„гҖӮжҜҸдёӘиў«з”ҹдә§е’Ңж¶Ҳиҙ№зҡ„жөҒйғҪдјҡиў«еҲҶй…ҚдёҖдёӘзј“еҶІжұ гҖӮзј“еҶІжұ з®ЎзҗҶиҖ…дёҖз»„зј“еҶІпјҲBufferпјүпјҢзј“еҶІеңЁиў«ж¶Ҳиҙ№еҗҺеҸҜд»Ҙиў«еӣһж”¶еҫӘзҺҜеҲ©з”ЁгҖӮ

    # 4. Flinkзҡ„еҸҚеҺӢжЈҖжөӢйҖ»иҫ‘дәҶи§Јеҗ—пјҹ

    Flink-1.13зүҲжң¬д№ӢеүҚпјҢдҪҝз”Ёзҡ„е Ҷж ҲйҮҮж ·ејҸж–№ејҸеҲӨж–ӯеҸҚеҺӢпјҢеңЁFlink-1.13зүҲжң¬ејҖе§ӢдҪҝз”ЁеҹәдәҺд»»еҠЎMailboxи®Ўж—¶зҡ„ж–№ејҸеҲӨж–ӯеҸҚеҺӢгҖӮ

    # 5. дёәд»Җд№Ҳж•°жҚ®еҖҫж–ңдјҡйҖ жҲҗеҸҚеҺӢпјҹ

    йҰ–е…ҲпјҢж•°жҚ®еҖҫж–ңжҳҜз”ұдәҺдёҚеҗҢзҡ„keyеҜ№еә”зҡ„ж•°жҚ®ж•°йҮҸдёҚеҗҢпјҢеҜјиҮҙдёҚеҗҢзҡ„taskжүҖеӨ„зҗҶзҡ„ж•°жҚ®йҮҸдёҚеҗҢзҡ„й—®йўҳгҖӮеҰӮжһңеңЁFlinkзӣёеҗҢзҡ„Taskзҡ„еӨҡдёӘSubTaskд№Ӣй—ҙпјҢдёӘеҲ«SubTaskжҺҘж”¶еҲ°зҡ„ж•°жҚ®йҮҸжҳҺжҳҫеӨ§дәҺе…¶д»–SubTaskжҺҘж”¶еҲ°зҡ„ж•°жҚ®йҮҸпјҢе°ұдјҡйҖ жҲҗж•°жҚ®еҖҫж–ңпјҢиҝҷж ·е°ұдјҡеҜјиҮҙд»»еҠЎжү§иЎҢиҝҮж…ўпјҢд»ҺиҖҢеј•иө·еҸҚеҺӢжғ…еҶөгҖӮ

    Flink Web UIеҸҜд»ҘзІҫзЎ®зҡ„зңӢеҲ°жҜҸдёӘSubTaskеӨ„зҗҶдәҶеӨҡе°‘ж•°жҚ®пјҢеҚіеҸҜеҲӨж–ӯеҮәFlinkд»»еҠЎжҳҜеҗҰеӯҳеңЁж•°жҚ®еҖҫж–ңгҖӮ

    # 6. еҸҚеҺӢжңүе“ӘдәӣеҚұе®іе‘ўпјҹ

    еҸҚеҺӢеҰӮжһңдёҚиғҪеҫ—еҲ°жӯЈеёёзҡ„еӨ„зҗҶпјҢеҸҜиғҪдјҡеҪұе“ҚеҲ°checkpointж—¶й•ҝе’ҢstateеӨ§е°ҸпјҢз”ҡиҮіеҸҜиғҪдјҡеҜјиҮҙиө„жәҗиҖ—е°Ҫз”ҡиҮізі»з»ҹеҙ©жәғгҖӮ

    • еҪұе“Қcheckpointж—¶й•ҝпјҡbarrierдёҚдјҡи¶ҠиҝҮжҷ®йҖҡж•°жҚ®пјҢж•°жҚ®еӨ„зҗҶйҳ»еЎһд№ҹдјҡеҜјиҮҙcheckpoint barrierжөҒз»Ҹж•ҙдёӘж•°жҚ®з®ЎйҒ“зҡ„ж—¶й—ҙеҸҳй•ҝпјҢеҜјиҮҙcheckpointжҖ»дҪ“ж—¶й—ҙпјҲEnd to End DurationпјүеҸҳй•ҝгҖӮ
    • еҪұе“ҚstateеӨ§е°ҸпјҡbarrierеҜ№йҪҗж—¶пјҢжҺҘеҸ—еҲ°иҫғеҝ«зҡ„иҫ“е…Ҙз®ЎйҒ“зҡ„barrierеҗҺпјҢе®ғеҗҺйқўзҡ„ж•°жҚ®дјҡиў«зј“еӯҳиө·жқҘдҪҶдёҚеӨ„зҗҶпјҢзӣҙеҲ°иҫғж…ўзҡ„иҫ“е…Ҙз®ЎйҒ“зҡ„barrierд№ҹеҲ°иҫҫпјҢиҝҷдәӣиў«зј“еӯҳзҡ„ж•°жҚ®дјҡиў«ж”ҫе…ҘеҲ°stateйҮҢйқўпјҢеҜјиҮҙcheckpointеҸҳеӨ§гҖӮ

    иҝҷдёӨдёӘеҪұе“ҚеҜ№дәҺз”ҹдә§зҺҜеўғзҡ„дҪңдёҡжқҘиҜҙжҳҜеҚҒеҲҶеҚұйҷ©зҡ„пјҢеӣ дёәcheckpointж—¶дҝқиҜҒж•°жҚ®дёҖиҮҙжҖ§зҡ„е…ій”®пјҢcheckpointж—¶й—ҙеҸҳй•ҝеҸҜиғҪеҜјиҮҙcheckpointи¶…ж—¶еӨұиҙҘпјҢиҖҢstateеӨ§е°ҸеҗҢж ·еҸҜиғҪжӢ–ж…ўcheckpointеҜјиҮҙOOMпјҲдҪҝз”ЁHeap-based StateBackendпјүжҲ–иҖ…зү©зҗҶеҶ…еӯҳдҪҝз”Ёи¶…еҮәе®№еҷЁиө„жәҗпјҲдҪҝз”ЁRocksDBStateBackendпјүзҡ„зЁіе®ҡжҖ§й—®йўҳгҖӮ

    # 7. Flinkдёӯзҡ„ж•°жҚ®еҖҫж–ңжңүе“Әдәӣи§ЈеҶіж–№жі•пјҹ

    • keyByеҗҺзҡ„иҒҡеҗҲж“ҚдҪңеӯҳеңЁж•°жҚ®еҖҫж–ңгҖӮдҪҝз”ЁLocalKeyByзҡ„жҖқжғіпјҢеңЁkeyByдёҠжёёз®—еӯҗж•°жҚ®еҸ‘йҖҒд№ӢеүҚпјҢйҰ–е…ҲеңЁдёҠжёёз®—еӯҗзҡ„жң¬ең°еҜ№ж•°жҚ®иҝӣиЎҢиҒҡеҗҲеҗҺпјҢеҶҚеҸ‘йҖҒеҲ°дёӢжёёпјҢдҪҝдёӢжёёжҺҘж”¶еҲ°зҡ„ж•°жҚ®йҮҸеӨ§еӨ§еҮҸе°‘пјҢд»ҺиҖҢдҪҝеҫ—keyByд№ӢеҗҺзҡ„иҒҡеҗҲж“ҚдҪңдёҚеҶҚжҳҜд»»еҠЎзҡ„з“¶йўҲгҖӮ
    • keyByд№ӢеүҚеҸ‘з”ҹж•°жҚ®еҖҫж–ңгҖӮдә§з”ҹиҜҘжғ…еҶөеҸҜиғҪжҳҜеӣ дёәж•°жҚ®жәҗзҡ„ж•°жҚ®жң¬иә«е°ұдёҚеқҮеҢҖпјҢдҫӢеҰӮз”ұдәҺжҹҗдәӣеҺҹеӣ kafkaзҡ„topicдёӯжҹҗдәӣpartitionзҡ„ж•°жҚ®йҮҸиҫғеӨ§пјҢжҹҗдәӣpartitionзҡ„ж•°жҚ®йҮҸиҫғе°‘гҖӮи§ЈеҶіж–№жі•пјҡйңҖиҰҒи®©Flinkд»»еҠЎејәеҲ¶иҝӣиЎҢshuffleгҖӮдҪҝз”ЁshuffleпјҢrebalanceжҲ–rescaleз®—еӯҗеҚіеҸҜе°Ҷж•°жҚ®еқҮеҢҖеҲҶй…ҚпјҢд»ҺиҖҢи§ЈеҶіж•°жҚ®еҖҫж–ңзҡ„й—®йўҳгҖӮ
    • keyByеҗҺзҡ„зӘ—еҸЈиҒҡеҗҲж“ҚдҪңеӯҳеңЁж•°жҚ®еҖҫж–ңгҖӮеӣ дёәдҪҝз”ЁдәҶзӘ—еҸЈпјҢеҸҳжҲҗдәҶжңүз•Ңж•°жҚ®пјҲж”’жү№пјүзҡ„еӨ„зҗҶпјҢзӘ—еҸЈй»ҳи®ӨжҳҜи§ҰеҸ‘ж—¶жүҚдјҡиҫ“еҮәдёҖжқЎз»“жһңеҸ‘йҖҒдёӢжёёпјҢжүҖд»ҘеҸҜд»ҘдҪҝз”ЁдёӨйҳ¶ж®өиҒҡеҗҲзҡ„ж–№ејҸгҖӮпјҲз¬¬дёҖйҳ¶ж®өпјҡkeyжӢјжҺҘйҡҸжңәж•°еүҚзјҖжҲ–еҗҺзјҖпјҢиҝӣиЎҢkeyByпјҢејҖзӘ—пјҢиҒҡеҗҲпјӣз¬¬дәҢйҳ¶ж®өпјҡжҢүз…§еҺҹжқҘзҡ„keyеҸҠwindowEndеҒҡkeyByпјҢиҒҡеҗҲпјү

    # 8. дёәд»Җд№ҲдҪҝз”ЁFlinkжӣҝд»ЈSparkпјҹ

    дҪҝз”ЁFlinkдё»иҰҒиҖғиҷ‘зҡ„жҳҜFlinkзҡ„дҪҺе»¶иҝҹгҖҒй«ҳеҗһеҗҗйҮҸе’ҢеҜ№жөҒејҸж•°жҚ®еә”з”ЁеңәжҷҜжӣҙеҘҪзҡ„ж”ҜжҢҒгҖӮеҸҰеӨ–пјҢFlinkеҸҜд»ҘеҫҲеҘҪзҡ„еӨ„зҗҶд№ұеәҸж•°жҚ®пјҢиҖҢдё”еҸҜд»ҘдҝқиҜҒexactly-onceзҡ„зҠ¶жҖҒдёҖиҮҙжҖ§гҖӮ

    # 9. еҰӮжһңдёӢзә§еӯҳеӮЁдёҚж”ҜжҢҒдәӢеҠЎпјҢFlinkжҳҜеҰӮжһңдҝқиҜҒexactly-onceзҡ„пјҹ

    з«ҜеҲ°з«Ҝзҡ„exactly-onceеҜ№sinkзҡ„иҰҒжұӮжҜ”иҫғй«ҳпјҢе…·дҪ“зҡ„е®һзҺ°дё»иҰҒжңүе№ӮзӯүеҶҷе…Ҙе’ҢдәӢеҠЎжҖ§еҶҷе…ҘдёӨз§Қж–№ејҸгҖӮе№ӮзӯүеҶҷе…Ҙзҡ„еңәжҷҜдҫқиө–дәҺдёҡеҠЎйҖ»иҫ‘пјҢжӣҙеёёи§Ғзҡ„жҳҜз”ЁдәӢеҠЎжҖ§еҶҷе…ҘгҖӮиҖҢдәӢеҠЎжҖ§еҶҷе…ҘеҸҲжңүйў„еҶҷж—Ҙеҝ—пјҲWALпјүе’ҢдёӨйҳ¶ж®өжҸҗдәӨпјҲ2PCпјүдёӨз§Қж–№ејҸгҖӮ

    еҰӮжһңеӨ–йғЁзі»з»ҹдёҚж”ҜжҢҒдәӢеҠЎпјҢйӮЈд№ҲеҸҜд»ҘдҪҝз”Ёйў„еҶҷж—Ҙеҝ—зҡ„ж–№ејҸпјҢжҠҠз»“жһңж•°жҚ®еҪ“жҲҗзҠ¶жҖҒдҝқеӯҳпјҢз„¶еҗҺеңЁж”¶еҲ°checkpointе®ҢжҲҗзҡ„йҖҡзҹҘж—¶пјҢдёҖж¬ЎжҖ§еҶҷе…Ҙsinkзі»з»ҹгҖӮ

    # 10. Flinkзҡ„зҠ¶жҖҒжңәеҲ¶жҳҜд»Җд№Ҳпјҹ

    # 10.1 Raw Stateе’ҢManagedState

    жҢүз…§з”ұFlinkз®ЎзҗҶиҝҳжҳҜз”ЁжҲ·з®ЎзҗҶпјҢзҠ¶жҖҒеҸҜд»ҘеҲҶдёәеҺҹе§ӢзҠ¶жҖҒпјҲRaw Stateпјүе’Ңжүҳз®ЎзҠ¶жҖҒпјҲManagedStateпјүгҖӮ

    • жүҳз®ЎзҠ¶жҖҒпјҲManagedStateпјүпјҡз”ұFlinkиҮӘиЎҢз®ЎзҗҶзҡ„stateгҖӮ
    • еҺҹе§ӢзҠ¶жҖҒпјҲRaw Stateпјүпјҡз”ұз”ЁжҲ·иҮӘиЎҢиҝӣиЎҢз®ЎзҗҶгҖӮ

    дёӨиҖ…зҡ„дё»иҰҒеҢәеҲ«жҳҜпјҡManaged Stateз”ұFlink Runtimeз®ЎзҗҶпјҢиҮӘеҠЁеӯҳеӮЁпјҢиҮӘеҠЁжҒўеӨҚпјҢеңЁеҶ…еӯҳз®ЎзҗҶдёҠжңүдјҳеҢ–пјӣиҖҢRaw StateйңҖиҰҒз”ЁжҲ·иҮӘе·ұз®ЎзҗҶпјҢйңҖиҰҒиҮӘе·ұеәҸеҲ—еҢ–пјӣManaged Stateж”ҜжҢҒе·ІзҹҘзҡ„ж•°жҚ®з»“жһ„пјҢжҜ”еҰӮValueпјҢListпјҢMapзӯүпјӣиҖҢRaw StateеҸӘж”ҜжҢҒеӯ—иҠӮж•°з»„пјҢжүҖжңүзҡ„зҠ¶жҖҒйғҪиҰҒиҪ¬жҚўдёәдәҢиҝӣеҲ¶еӯ—иҠӮж•°з»„жүҚеҸҜд»ҘгҖӮ

    # 10.2 KeyedStateе’ҢOperatorState

    StateжҢүз…§жҳҜеҗҰжңүkeyеҲ’еҲҶдёәKeyedStateе’ҢOperatorStateдёӨз§ҚгҖӮ

    KeyedStateеҸӘиғҪз”ЁеңЁkeyedStreamзҡ„з®—еӯҗдёҠпјҢзҠ¶жҖҒе’Ңзү№е®ҡзҡ„keyз»‘е®ҡгҖӮkeyedStreamжөҒдёҠзҡ„жҜҸдёҖдёӘkeyеҜ№еә”дёҖдёӘstateеҜ№иұЎгҖӮиӢҘдёҖдёӘoperatorеӨ„зҗҶеӨҡдёӘkeyпјҢи®ҝй—®зӣёеә”зҡ„зҡ„еӨҡдёӘstateпјҢеҸҜеҜ№еә”еӨҡдёӘstateгҖӮkeyedStateдҝқеӯҳеңЁStateBackendдёӯпјҢйҖҡиҝҮRuntimeContextи®ҝй—®пјҢе®һзҺ°Rich FunctionжҺҘеҸЈгҖӮж”ҜжҢҒеӨҡз§Қж•°жҚ®з»“жһ„пјҡValueStateпјҢListStateпјҢReducingStateпјҢAggregatingStateпјҢMapStateгҖӮ

    OperatorStateеҸҜд»Ҙз”ЁдәҺжүҖжңүз®—еӯҗпјҢдҪҶж•ҙдёӘз®—еӯҗеҸӘеҜ№еә”дёҖдёӘstateгҖӮOperatorStateе®һзҺ°CheckpointedFunctionжҲ–иҖ…ListCheckpointedпјҲе·Із»ҸиҝҮж—¶пјүжҺҘеҸЈгҖӮзӣ®еүҚеҸӘж”ҜжҢҒListStateе’ҢBroadCastStateж•°жҚ®з»“жһ„гҖӮ

    # 10.3 еӯҳеӮЁ

    еңЁFlinkдёӯпјҢstateдёҖиҲ¬иў«еӯҳеӮЁеңЁStateBackendйҮҢйқўгҖӮжҖ»е…ұеҢ…еҗ«дёүз§Қж–№ејҸпјҡеҶ…еӯҳпјҢж–Үд»¶пјҢRocksDBзӯүгҖӮ

    • MemoryStateBackendпјҡиҝҗиЎҢж—¶жүҖйңҖзҡ„stateж•°жҚ®е…ЁйғЁдҝқеӯҳеңЁTaskManager JVMе ҶеҶ…еӯҳдёҠпјҢжү§иЎҢжЈҖжҹҘзӮ№зҡ„ж—¶еҖҷпјҢдјҡжҠҠstateзҡ„еҝ«з…§ж•°жҚ®дҝқеӯҳеңЁJobManagerиҝӣзЁӢзҡ„еҶ…еӯҳдёӯгҖӮеҹәдәҺеҶ…еӯҳзҡ„StateBackendеңЁз”ҹдә§зҺҜеўғдёӢдёҚе»әи®®дҪҝз”ЁпјҢеҸҜд»ҘеңЁжң¬ең°ејҖеҸ‘и°ғиҜ•гҖӮStateеӯҳеӮЁеңЁJobManagerзҡ„еҶ…еӯҳдёӯпјҢеҸ—йҷҗдәҺJobManagerзҡ„еҶ…еӯҳеӨ§е°ҸпјӣжҜҸдёӘStateй»ҳи®Ө5MBпјҢеҸҜйҖҡиҝҮMemoryStateBackendжһ„йҖ еҮҪж•°и°ғж•ҙпјӣжҜҸдёӘStateдёҚиғҪи¶…иҝҮAkka FrameеӨ§е°ҸгҖӮ
    • FSStateBackendпјҡиҝҗиЎҢж—¶жүҖйңҖзҡ„Stateж•°жҚ®е…ЁйғЁдҝқеӯҳеңЁTaskManagerзҡ„еҶ…еӯҳдёӯпјҢжү§иЎҢжЈҖжҹҘзӮ№зҡ„ж—¶еҖҷпјҢдјҡжҠҠStateзҡ„еҝ«з…§ж•°жҚ®дҝқеӯҳеңЁй…ҚзҪ®зҡ„ж–Үд»¶зі»з»ҹдёӯгҖӮйҖӮз”ЁдәҺеӨ„зҗҶе°ҸзҠ¶жҖҒпјҢзҹӯзӘ—еҸЈпјҢжҲ–иҖ…е°Ҹй”®еҖјзҠ¶жҖҒзҡ„жңүзҠ¶жҖҒеӨ„зҗҶд»»еҠЎпјӣStateйҰ–е…Ҳиў«еӯҳеӮЁеңЁTaskManagerзҡ„еҶ…еӯҳдёӯпјӣStateеӨ§е°ҸдёҚиғҪи¶…иҝҮTaskManagerзҡ„еҶ…еӯҳпјӣTaskManagerејӮжӯҘе°ҶStateж•°жҚ®еҶҷе…ҘеҲ°еӨ–йғЁеӯҳеӮЁгҖӮ
    • RocksDBStateBackendпјҡдҪҝз”ЁеөҢе…ҘејҸзҡ„жң¬ең°ж•°жҚ®еә“RocksDBе°ҶжөҒи®Ўз®—ж•°жҚ®зҠ¶жҖҒеӯҳеӮЁеңЁжң¬ең°зЈҒзӣҳдёӯгҖӮеңЁжү§иЎҢжЈҖжҹҘзӮ№зҡ„ж—¶еҖҷпјҢеҶҚе°Ҷж•ҙдёӘRocksDBдёӯдҝқеӯҳзҡ„Stateж•°жҚ®е…ЁйҮҸжҲ–иҖ…еўһйҮҸжҢҒд№…еҢ–еҲ°й…ҚзҪ®зҡ„ж–Үд»¶зі»з»ҹдёӯгҖӮжңҖйҖӮеҗҲз”ЁдәҺеӨ„зҗҶеӨ§зҠ¶жҖҒпјҢй•ҝзӘ—еҸЈпјҢжҲ–иҖ…еӨ§й”®еҖјзҠ¶жҖҒзҡ„жңүзҠ¶жҖҒеӨ„зҗҶд»»еҠЎпјӣRocksDBStateBackendйқһеёёйҖӮеҗҲз”ЁдәҺй«ҳеҸҜз”Ёж–№жЎҲпјӣRocksDBStateBackendжҳҜзӣ®еүҚе”ҜдёҖж”ҜжҢҒеўһйҮҸжЈҖжҹҘзӮ№зҡ„еҗҺз«ҜгҖӮеўһйҮҸжЈҖжҹҘзӮ№йқһеёёйҖӮз”ЁдәҺи¶…еӨ§зҠ¶жҖҒзҡ„еңәжҷҜгҖӮ

    # 11. жө·йҮҸkeyжҖҺд№ҲеҺ»йҮҚпјҹ

    дҪҝз”Ёзұ»дјјдәҺScalaзҡ„setж•°жҚ®з»“жһ„жҲ–иҖ…redisзҡ„setжҳҫз„¶жҳҜдёҚиЎҢзҡ„пјҢеӣ дёәеҸҜиғҪжңүдёҠдәҝдёӘkeyпјҢеҶ…еӯҳж”ҫдёҚдёӢгҖӮжүҖд»ҘеҸҜд»ҘиҖғиҷ‘дҪҝз”ЁеёғйҡҶиҝҮж»ӨеҷЁпјҲBloom FilterпјүжқҘеҺ»йҮҚгҖӮ

    # 12. Flinkзҡ„checkpointжңәеҲ¶еҜ№жҜ”sparkжңүд»Җд№ҲдёҚеҗҢе’ҢдјҳеҠҝпјҹ

    spark streamingзҡ„checkpointд»…д»…жҳҜй’ҲеҜ№driverзҡ„ж•…йҡңжҒўеӨҚеҒҡдәҶж•°жҚ®е’Ңе…ғж•°жҚ®зҡ„checkpointгҖӮиҖҢflinkзҡ„checkpointжңәеҲ¶иҰҒеӨҚжқӮеҫҲеӨҡпјҢе®ғйҮҮз”Ёзҡ„жҳҜиҪ»йҮҸзә§зҡ„еҲҶеёғејҸеҝ«з…§пјҢе®һзҺ°дәҶжҜҸдёӘз®—еӯҗзҡ„еҝ«з…§пјҢеҸҠжөҒеҠЁдёӯзҡ„ж•°жҚ®зҡ„еҝ«з…§гҖӮ

    # 13. иҜҰз»Ҷи§ЈйҮҠдёҖдёӢFlinkзҡ„watermarkжңәеҲ¶пјҹ

    Watermarkзҡ„жң¬иҙЁжҳҜFlinkдёӯиЎЎйҮҸEventTimeиҝӣеұ•зҡ„дёҖдёӘжңәеҲ¶пјҢдё»иҰҒз”ЁжқҘеӨ„зҗҶд№ұеәҸзҡ„ж•°жҚ®гҖӮ

    # 14. Flinkдёӯзҡ„exactly-onceиҜӯд№үеҰӮдҪ•е®һзҺ°зҡ„пјҢзҠ¶жҖҒеҰӮдҪ•еӯҳеӮЁзҡ„пјҹ

    Flinkдҫқйқ checkpointжңәеҲ¶жқҘе®һзҺ°exactly-onceиҜӯд№үпјҢеҰӮжһңиҰҒе®һзҺ°з«ҜеҲ°з«Ҝзҡ„exactly-onceпјҢиҝҳйңҖиҰҒеӨ–йғЁsourceе’Ңsinkж»Ўи¶ідёҖе®ҡзҡ„жқЎд»¶гҖӮзҠ¶жҖҒзҡ„еӯҳеӮЁйҖҡиҝҮзҠ¶жҖҒеҗҺз«ҜжқҘз®ЎзҗҶпјҢFlinkдёӯеҸҜд»Ҙй…ҚзҪ®дёҚеҗҢзҡ„зҠ¶жҖҒеҗҺз«ҜгҖӮ

    # 15. Flink CEPзј–зЁӢдёӯеҪ“зҠ¶жҖҒжІЎжңүеҲ°иҫҫзҡ„ж—¶еҖҷдјҡе°Ҷж•°жҚ®дҝқеӯҳеҲ°е“ӘйҮҢпјҹ

    еңЁжөҒејҸеӨ„зҗҶдёӯпјҢCEPеҪ“з„¶жҳҜиҰҒж”ҜжҢҒEventTimeзҡ„пјҢйӮЈд№ҲзӣёеҜ№еә”зҡ„д№ҹиҰҒж”ҜжҢҒж•°жҚ®зҡ„иҝҹеҲ°зҺ°иұЎпјҢд№ҹе°ұжҳҜиҜҙwatermarkзҡ„еӨ„зҗҶйҖ»иҫ‘гҖӮCEPеҜ№жңӘеҢ№й…ҚжҲҗеҠҹзҡ„дәӢд»¶еәҸеҲ—зҡ„еӨ„зҗҶпјҢе’ҢиҝҹеҲ°ж•°жҚ®жҳҜзұ»дјјзҡ„гҖӮеңЁFlink CEPзҡ„еӨ„зҗҶйҖ»иҫ‘дёӯпјҢзҠ¶жҖҒжІЎжңүж»Ўи¶іе’ҢиҝҹеҲ°зҡ„ж•°жҚ®пјҢйғҪдјҡеӯҳеӮЁеңЁдёҖдёӘMapж•°жҚ®з»“жһ„дёӯпјҢд№ҹе°ұжҳҜиҜҙпјҢеҰӮжһңжҲ‘д»¬йҷҗе®ҡеҲӨж–ӯдәӢд»¶еәҸеҲ—зҡ„ж—¶й•ҝдёә5еҲҶй’ҹпјҢйӮЈд№ҲеҶ…еӯҳдёӯе°ұдјҡеӯҳеӮЁ5еҲҶй’ҹзҡ„ж•°жҚ®пјҢиҝҷд№ҹжҳҜеҜ№еҶ…еӯҳзҡ„жһҒеӨ§жҚҹдјӨд№ӢдёҖгҖӮ
